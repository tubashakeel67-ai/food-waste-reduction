# 🥗 AI Food Waste Reduction System

An end-to-end Machine Learning system that predicts food waste risk 
and forecasts meal demand to help reduce food waste.

## 🚀 Tech Stack
| Layer | Technology |
|---|---|
| ML Models | Random Forest, XGBoost |
| Backend API | FastAPI + SQLite |
| Frontend | Streamlit |
| Deployment | Render + Streamlit Cloud |

## ⚡ Quick Start (3 commands)
```bash
git clone https://github.com/YOUR_USERNAME/food-waste-reduction.git
cd food-waste-reduction
pip install -r requirements.txt
```

## 📊 Features
- Waste risk prediction per food item
- Weekly demand forecasting by meal type
- Real-time expiry alerts dashboard
- REST API with 3 endpoints
